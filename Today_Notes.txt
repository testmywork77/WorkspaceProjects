Docker:
	Step 1: Install "Docker Desktop Installer"
		Open https://www.docker.com/get-started
		Download "Docker Desktop" -Choose Download for Windows
		Install Docker Desktop Installer
		After install restart the Machine
	Step 2: Install Linux Kernel for Windows 11
	Step 3 - Download the Linux kernel update package 
		Open https://docs.microsoft.com/en-us/windows/wsl/install-manual#step-4---download-the-linux-kernel-update-package
		4.1 Download the latest package: WSL2 Linux kernel update package for x64 machines
		4.2 Run the update package downloaded in the previous step. (Double-click to run - you will be prompted for elevated permissions, select ‘yes’ to approve this installation.)

	To run sample Docker Images, open Windows Terminal/Gitbash:
	1. Run the image "docker/getting-started"
		$ docker run -d -p 80:80 docker/getting-started
	Open the browser and check the above web application.  i.e.	http://localhost:80
